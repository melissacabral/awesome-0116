== Description ==
Awesome is a theme designed to teach the basics of theme design and development. It has a straightforward two-column layout that is well suited to blogs and business websites.  It has custom color options, a fluid grid using a mobile-first approach, and support for a product catalog. 

* Mobile-first, Responsive Layout
* Custom Colors
* Custom Header
* Social Links
* Post Formats

For more information, please go to https://wordpress.melissacabral.com.

== Installation ==

1. In your admin panel, go to Appearance -> Themes and click the 'Add New' button.
2. Click the Upload button. Browse for the awesome.zip file
3. Click on the 'Activate' button to use your new theme right away.
5. Navigate to Appearance > Customize in your admin panel and customize to taste.
